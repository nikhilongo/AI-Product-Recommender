# 🪑 AI Furniture Recommendation Web App

An intelligent furniture recommendation system that uses **AI-powered multimodal retrieval** and **generative creativity** to recommend furniture products based on user queries or images.

---

## 🚀 Overview

This project combines **retrieval-augmented generation (RAG)** and **multimodal AI** (text + image) to provide intelligent furniture recommendations.

Users can:
- Enter a **text query** (e.g., “chai”, “modern sofa”, “study chair”)
- Or upload / provide an **image URL** of furniture inspiration  
And the system will:
1. Retrieve similar products from the database.
2. Generate a **creative and user-friendly product description** using **Google Gemini AI**.

---

## 🧠 Features

- 🔍 **Semantic Search** — Uses Sentence Transformers and CLIP to find visually and semantically similar products.
- 🧩 **Multimodal RAG** — Combines text and image embeddings for more accurate results.
- ✍️ **AI Descriptions** — Generates human-like, persuasive recommendations.
- ⚡ **FastAPI Backend** — Lightweight, high-performance Python API.
- 💅 **React Frontend** — Modern, responsive, and easy-to-use UI.
